# Percadyn

Percadyn (PERmutation Cellular Automata DYNamics) is a browser-based generative music application built with F#, Fable, and Feliz.

## Context

It uses Time-Symmetric Sequencers based on the Margolus Neighborhood principle.

## Tech Stack

- **Language**: F#
- **Transpiler**: Fable 4
- **UI**: Feliz
- **Bundler**: Vite

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   dotnet restore
   ```

2. Run development server:

   ```bash
   npm run dev
   ```
